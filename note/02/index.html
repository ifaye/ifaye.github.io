如何搭建本地服务器

开发的时候经常都是在本地浏览器查看效果，但有时需要在本地其他设备例如手机查看效果。 这个时候就需要搭建一个本地服务器来跑代码，用过的有下面三种方式来实现。

1) Apache Tomcat

首先是按装 jdk，然后是配置环境变量，然后就是运行 Tomcat。

具体步骤参照我的笔记

2) WampServer

WampServer 就是 Windows Apache Mysql PHP 集成安装环境，免去了配置环境，打开就用。

具体步骤参照我的笔记

3) http-server

如果电脑装了 node.js，那么可以运用 npm 的一个命令服务器 http-server

具体步骤参照我的笔记
