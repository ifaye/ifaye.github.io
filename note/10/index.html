Bootstrap 是全球最受欢迎的前端框架

以前就学习和使用过 bootstrap，只不过那时候还是 bootstrap3，而且也没做什么笔记。现在刚好做一下笔记，免得遗忘。bootstrap4功能更加强大，提供了许多很常用的组件。如果要兼容 ie8-ie10 建议还是使用 bootstrap3 ,如果不需要则可以用 bootstrap4

首先是框架 css 和 js 的引用。可以下载源文件，也可以用 CDN 外联。记得先 jquery，再 popper，再 bootstrap。有些组件需要依赖 jq,有些组件需要依赖 popper。

以下是需要依赖 jquery 的组件

  1. alerts 警告框
  2. Buttons checkbox/radio 切换按钮和单选多选
  3. 所有滑动行为、控件和指示器
  4. 切换显示操作如 tab 之类
  5. 用于显示和定位的下拉框 (同样要依赖 Popper)
  6. modal 模态框(弹窗)
  7. navbar 复杂导航条
  8. Tooltips 工具提示框 popovers 弹出框 (同样要依赖 Popper)
  9. scrollspy 滚动监听

必须要有的 meta 标签

 <meta charset="utf-8">
 <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">

全局 box-size 都是 border-box

布局，栅格系统和页面内容需要包裹在 .container 容器(固定宽度，支持响应式)。.container-fluid 用于 100% 宽度，占据全部视口（viewport）的容器。 行 row 必须包含在 .container 或 .container-fluid中，一行最多分为 12 份。
